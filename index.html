FaB Boost for Images

FaB Boost for Images is a lightweight, efficient image processing microservice designed to simplify remote image hosting for client sites. It provides a straightforward, URL-based API for hassle-free image transformations.

With FaB Boost for Images, generating responsive <img> tags is easy‚Äîcomplete with srcset attributes and scaled images, without relying on complex frameworks or hefty servers. Images are remotely downloaded, cached, and optimized for all screen sizes, making it a practical solution for scalable image delivery across client projects.

How It Works

  • Remote Image Download and Caching: Images can be requested via URLs like
    img.my-domain.com/client-domain.com/photos/cat.jpg?w=100 . Images are downloaded from trusted client domains and cached for faster access.
  • On-the-Fly Image Transformations: Supports resizing, cropping, blurring, and more‚Äîall transformations are cached to ensure efficient delivery.
  • Recommended Setup: Use a service like Cloudflare (even the free plan) on img.my-domain.com for enhanced performance.
